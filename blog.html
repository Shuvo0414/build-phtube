1. Discuss the scope of var, let, and const
var: It has function and global scope. It can be redeclared, and it can also be reassigned.
let: It has block scope. It cannot be redeclared, but it can be reassigned.
const: It has block scope. It cannot be redeclared, and it cannot be reassigned.

2. The use cases of null and undefined.
null : Used when there's no value intentionally set, indicating the absence of a value.
undefined : When there's no value assigned or defined, indicating the absence of an assigned value.

3. What do you mean by REST API?
The meaning of REST API is Representational State Transfer Application Programming Interface. It uses HTTP methods like GET, POST, PUT, DELETE, similar to how they are used in web services. A set of rules and conventions are followed to create and interact with web services. These are designed to be simple, scalable, and stateless, making them a popular choice for web and mobile applications to access and manipulate data on remote servers.
